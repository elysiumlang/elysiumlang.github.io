console — Unified Logging

Standard logging with level-specific prefixes. Works in both the compiled binary (via printf) and the JS runtime (via native console.*).

console.debug(msg: String) -> Nil

Log a debug message with [DEBUG] prefix.

console.info(msg: String) -> Nil

Log an info message with [INFO] prefix.

console.warn(msg: String) -> Nil

Log a warning with [WARN] prefix.

console.error(msg: String) -> Nil

Log an error with [ERROR] prefix.

console.log(msg: String) -> Nil

Log a general message with [LOG] prefix.

print(msg: String) -> Nil

Bare print without any prefix. Always available.

func main() {
    console.debug("debug message")
    console.info("info message")
    console.warn("warn message")
    console.error("error message")
    console.log("log message")
    print("bare print")

    // With string interpolation
    let name = "Elysium"
    console.info("Hello, ", name)
}

fs — Filesystem Operations

Read, write, and manipulate files and directories. Backend uses C stdlib (fopen, fgets, etc.), JS runtime uses Node's fs module.

fs.readFile(path: String) -> String

Read entire file contents as a string.

fs.writeFile(path: String, content: String) -> Nil

Write content to a file (overwrites if exists).

fs.appendFile(path: String, content: String) -> Nil

Append content to the end of a file.

fs.exists(path: String) -> Bool

Check if a file or directory exists.

fs.copyFile(src: String, dst: String) -> Nil

Copy a file from source to destination.

fs.rename(old: String, new: String) -> Nil

Rename or move a file.

fs.removeFile(path: String) -> Nil

Delete a file.

fs.createDir(path: String) -> Nil

Create a new directory.

fs.removeDir(path: String) -> Nil

Remove an empty directory.

func main() {
    fs.writeFile("hello.txt", "Hello from Elysium!")
    let content = fs.readFile("hello.txt")
    print(content)

    if fs.exists("hello.txt") then
        fs.copyFile("hello.txt", "backup.txt")

    fs.removeFile("hello.txt")
}

transport — Networking

HTTP, WebSocket, and MQTT networking utilities. In the C backend, HTTP uses curl via popen. The JS runtime uses native fetch, WebSocket, and mqtt.js.

HTTP

transport.get(url: String) -> String

Perform an HTTP GET request.

transport.post(url: String, body: String) -> String

Perform an HTTP POST request with a body.

transport.put(url: String, body: String) -> String

Perform an HTTP PUT request.

transport.delete(url: String) -> String

Perform an HTTP DELETE request.

WebSocket

transport.wsConnect(url: String) -> String

Connect to a WebSocket server. Returns a connection handle.

transport.wsSend(conn: String, data: String) -> Nil

Send data over an open WebSocket connection.

transport.wsClose(conn: String) -> Nil

Close a WebSocket connection.

MQTT

transport.mqttConnect(broker: String, clientId: String) -> String

Connect to an MQTT broker. Returns a client handle.

transport.mqttPublish(client: String, topic: String, msg: String) -> Nil

Publish a message to an MQTT topic.

transport.mqttSubscribe(client: String, topic: String) -> Nil

Subscribe to an MQTT topic.

transport.mqttDisconnect(client: String) -> Nil

Disconnect from an MQTT broker.

// HTTP
let data = transport.get("https://api.example.com/data")

// WebSocket
let ws = transport.wsConnect("wss://echo.example.com")
transport.wsSend(ws, "hello from Elysium")
transport.wsClose(ws)

// MQTT
let client = transport.mqttConnect("mqtt://broker.example.com", "elysium-client")
transport.mqttPublish(client, "topic/hello", "world")
transport.mqttDisconnect(client)

regex — Regular Expressions

Pattern matching with POSIX extended regular expressions. Backend uses C regcomp/regexec, JS runtime wraps native RegExp.

regex.test(pattern: String, str: String) -> Bool

Test if the pattern matches anywhere in the string.

regex.match(pattern: String, str: String) -> String

Extract the first match.

regex.search(pattern: String, str: String) -> Int

Find the starting position of the first match.

regex.replace(pattern: String, str: String, replacement: String) -> String

Replace the first occurrence of pattern.

regex.split(pattern: String, str: String) -> String

Split string by pattern occurrences (returns comma-joined for now).

func main() {
    regex.test("\\d+", "hello 123 world")
    regex.match("\\d+", "abc 456 def")
    regex.search("\\d+", "abc 789 def")
    regex.replace("foo", "hello foo world", "bar")
    regex.split(",", "a,b,c")
}

datetime — Date & Time

Unified date/time APIs across backend (C time.h) and frontend (JS Date). All APIs use unix timestamps (seconds since epoch, Int) as the universal bridge.

Core Functions

datetime.now() -> Int

Current unix timestamp (seconds since epoch).

datetime.fromTimestamp(ts: Int) -> String

Convert unix timestamp to a locale-dependent string.

datetime.format(ts: Int, fmt: String) -> String

Format timestamp using strftime-style format codes: %Y %m %d %H %M %S %a %b.

datetime.parse(str: String, fmt: String) -> Int

Parse a formatted string back to timestamp (JS runtime only).

Component Extraction

datetime.year(ts: Int) -> Int

Extract the year component.

datetime.month(ts: Int) -> Int

Extract the month (1-12).

datetime.day(ts: Int) -> Int

Extract the day of month (1-31).

datetime.hour(ts: Int) -> Int

Extract the hour (0-23).

datetime.minute(ts: Int) -> Int

Extract the minute (0-59).

datetime.second(ts: Int) -> Int

Extract the second (0-59).

datetime.weekday(ts: Int) -> Int

Extract the day of week (0=Sunday, 6=Saturday).

Arithmetic

datetime.addDays(ts: Int, n: Int) -> Int

Add n days to a timestamp.

datetime.addHours(ts: Int, n: Int) -> Int

Add n hours to a timestamp.

datetime.diffSeconds(ts1: Int, ts2: Int) -> Int

Difference between two timestamps in seconds.

func main() {
    let now = datetime.now()
    let formatted = datetime.format(now, "%Y-%m-%d %H:%M:%S")
    print(formatted)

    let tomorrow = datetime.addDays(now, 1)
    let diff = datetime.diffSeconds(tomorrow, now)
    print("Seconds until tomorrow: ", diff)

    // Extract components
    let y = datetime.year(now)
    let m = datetime.month(now)
    let d = datetime.day(now)
}

dict — Key-Value Dictionaries

Mutable string-keyed dictionaries for key-value storage.

dict.create() -> String

Create a new empty dictionary (returns a handle).

dict.set(dictId: String, key: String, value: String) -> Nil

Set a key-value pair.

dict.get(dictId: String, key: String) -> String

Get a value by key.

dict.has(dictId: String, key: String) -> Bool

Check if a key exists.

dict.delete(dictId: String, key: String) -> Nil

Remove a key-value pair.

dict.keys(dictId: String) -> String

Get all keys as a comma-separated string.

dict.length(dictId: String) -> Int

Get the number of entries.

dict.clear(dictId: String) -> Nil

Remove all entries.

func main() {
    let d = dict.create()
    dict.set(d, "name", "Alice")
    dict.set(d, "age", "30")
    print(dict.get(d, "name"))
    dict.delete(d, "age")
    print(dict.length(d))
    dict.clear(d)
}

json — JSON Operations

Parse, construct, and query JSON data.

json.parse(str: String) -> String

Parse a JSON string into an internal representation handle.

json.stringify(handle: String) -> String

Serialize an internal handle back to a JSON string.

json.get(handle: String, path: String) -> String

Get a value by dot-notation path (e.g. "choices.0.message.content").

json.buildObject(pairs...) -> String

Build a JSON object from alternating key-value pairs (used by inline JSON literals).

json.buildArray(items...) -> String

Build a JSON array from items.

json.free(handle: String) -> Nil

Free an internal JSON handle.

// Inline JSON (desugars to json.buildObject)
let body = { "model": "gpt-4o", "temperature": "0.7" }

// Parse and query
let data = json.parse(`{"choices": [{"message": {"content": "Hello"}}]}`)
let content = json.get(data, "choices.0.message.content")
json.free(data)

http — HTTP Request Utilities

Low-level HTTP request/response utilities. For common methods (GET/POST/PUT/DELETE), use transport.* instead.

http.request(method: String, url: String, headers: String, body: String) -> String

Send an HTTP request with custom method, headers (JSON), and body. Returns response body.

func main() {
    let headers = `{"Authorization": "Bearer token123"}`
    let body = `{"query": "hello"}`
    let response = http.request("POST", "https://api.example.com/chat", headers, body)
}

auth — Authentication & Authorization

Comprehensive auth operations: JWT, sessions, password hashing, OAuth2, passkeys, RBAC, multi-tenant.

JWT

auth.jwtSign(payload: String, expiresIn: String) -> String

Create a signed JWT with optional expiration ("1h", "7d", etc.).

auth.jwtVerify(token: String) -> String

Verify and decode a JWT.

auth.jwtDecode(token: String) -> String

Decode a JWT without verification.

Sessions & Passwords

auth.createSession(userId: String, data: String) -> String

Create a session and return session ID.

auth.getSession(sessionId: String) -> String

Get session data.

auth.destroySession(sessionId: String) -> Nil

Destroy a session.

auth.hashPassword(password: String) -> String

Hash a password with PBKDF2-SHA256.

auth.verifyPassword(password: String, hash: String) -> String

Verify a password against its hash.

func main() {
    // JWT
    let token = auth.jwtSign(`{"sub": "user123"}`, "1h")
    let verified = auth.jwtVerify(token)

    // Password hashing
    let hash = auth.hashPassword("my-secure-password")
    let match = auth.verifyPassword("my-secure-password", hash)
}

Additional functions: checkPermission, hasRole, checkAccess, setRoles, oauth2Authorize, oauth2Token, passkeyRegister, passkeyAuthenticate, generateApiKey, validateApiKey, tenantContext, createTenant, grantRole, revokeRole.

langchain — LLM & AI Operations

LangChain-style AI operations for LLM calls, RAG, summarization, classification, translation, and agents. Installable via EPM: epm install @elysium/langchain.

langchain.llm(model: String, prompt: String) -> String

Send a prompt to an LLM and get a response.

langchain.chat(model: String, messages: String, temperature: String) -> String

Multi-turn chat with message history.

langchain.embed(text: String) -> String

Generate an embedding for text.

langchain.summarize(text: String) -> String

Summarize long text.

langchain.rag(query: String, context: String) -> String

Retrieval-augmented generation.

langchain.agent(model: String, systemPrompt: String, userMessage: String) -> String

Agent with system prompt and user message.

Uses environment variables: OPENAI_BASE_URL (default: http://localhost:11434/v1 for Ollama), LLM_MODEL, OPENAI_API_KEY.

import "#/@elysium/langchain" as langchain

func main() {
    let response = langchain.llm("gpt-4o", "Hello!")
    let summary = langchain.summarize("Long text here...")
}

langgraph — Stateful Agent Graphs

LangGraph-style stateful agent orchestration with graphs, nodes, edges, and conditional routing. Installable via EPM: epm install @elysium/langgraph.

langgraph.graph(name: String) -> String

Create a new state graph.

langgraph.addNode(graphId: String, name: String, action: String) -> Nil

Add a node to the graph.

langgraph.addEdge(graphId: String, from: String, to: String) -> Nil

Connect two nodes with an edge.

langgraph.compile(graphId: String) -> String

Compile the graph into an executable pipeline.

langgraph.invoke(graphId: String, input: String) -> String

Run the graph with input and get output.

langgraph.stream(graphId: String, input: String) -> String

Run the graph and stream intermediate outputs.

import "#/@elysium/langgraph" as langgraph

func main() {
    let gid = langgraph.graph("simple_agent")
    langgraph.addNode(gid, "input", "function node")
    langgraph.addEdge(gid, "input", "output")
    let compiled = langgraph.compile(gid)
    let result = langgraph.invoke(gid, "idle")
}

Bluetooth Low Energy operations: scanning, connection, characteristic read/write. Import via import "#/@elysium/ble" as ble.

Start scanning for BLE devices.

ble.connect(address: String) -> String

Connect to a BLE device by address.

Disconnect from a device.

Read a characteristic value.

ble.writeCharacteristic(deviceId: String, serviceUuid: String, charUuid: String, value: String) -> Nil

import "#/@elysium/ble" as ble

func main() {
    let deviceId = ble.scan()
    ble.connect(deviceId)
    let value = ble.readCharacteristic(deviceId, "180D", "2A37")

}

zigbee.off(deviceId: String, endpoint: Int, cluster: String) -> Nil
