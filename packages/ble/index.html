@elysium/ble 0.1.0

Bluetooth Low Energy operations — scanning, connecting, reading/writing characteristics, RSSI, and notifications.

epm install @elysium/ble

Import

import "#/@elysium/ble" as ble

API

scan() -> StringStart scanning for BLE devices
stopScan()Stop scanning
connect(address) -> StringConnect to a device, returns device ID
disconnect(deviceId)Disconnect from a device
readCharacteristic(deviceId, serviceUuid, charUuid) -> StringRead a characteristic value
writeCharacteristic(deviceId, serviceUuid, charUuid, value)Write a characteristic value
readRssi(deviceId) -> StringRead RSSI (signal strength)
deviceName(deviceId) -> StringGet the friendly name of a device
isConnected(deviceId) -> StringCheck if a device is connected
isScanning() -> StringCheck if currently scanning
