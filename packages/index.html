Installing Packages

Packages are installed via the Elysium Package Manager (EPM), which is bundled with the compiler.

Install from registry

elysium install @elysium/langchain

Install a specific version

elysium install @elysium/auth@0.1.0

Install from workspace

If a package is available in your workspace's packages/ directory, import it directly:

import "#/@elysium/langchain" as langchain

Create a new package

elysium init my-package

Publish a package

From the package directory:

elysium publish
