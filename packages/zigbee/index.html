@elysium/zigbee 0.1.0

Zigbee Home Automation — device control, groups, scenes, network management, and bindings.

epm install @elysium/zigbee

Import

import "#/@elysium/zigbee" as zigbee

API

start()Start the Zigbee coordinator
shutdown()Shutdown the coordinator
permitJoin(seconds)Permit devices to join for N seconds
scan()Scan for nearby Zigbee devices
on(deviceId, endpoint, cluster)Turn a device on
off(deviceId, endpoint, cluster)Turn a device off
toggle(deviceId, endpoint, cluster)Toggle a device
readAttribute(deviceId, endpoint, cluster, attribute) -> StringRead a Zigbee attribute
writeAttribute(deviceId, endpoint, cluster, attribute, value)Write a Zigbee attribute
addToGroup(deviceId, groupId)Add device to a group
removeFromGroup(deviceId, groupId)Remove device from a group
bind(sourceDevId, sourceEp, destDevId, destEp)Bind two Zigbee endpoints
getDeviceName(deviceId) -> StringGet device friendly name
getManufacturer(deviceId) -> StringGet device manufacturer
getDeviceCount() -> IntGet number of known devices
getPanId() -> IntGet PAN ID
getChannel() -> IntGet current channel
isJoined() -> StringCheck if coordinator is on a network
isOnline(deviceId) -> StringCheck if a device is online
isPermittingJoin() -> StringCheck if join is permitted
