@elysium/langchain 0.1.0

Elysium port of LangChain by LangChain Inc — LLM calls, chains, prompts, and tool integration.

epm install @elysium/langchain

Import

import "#/@elysium/langchain" as langchain

API

llm(model, prompt) -> StringCall an LLM with a prompt
chat(model, system, message) -> StringChat with system prompt and user message
embed(text) -> StringGet text embeddings
similarity(a, b) -> StringCompute similarity between two texts
template(templateStr, vars) -> StringRender a prompt template with variables
rag(query, context) -> StringRetrieval-augmented generation
summarize(text) -> StringSummarize text
analyze(text, instruction) -> StringAnalyze text with custom instruction
classify(text, labels) -> StringClassify text into given labels
translate(text, targetLang) -> StringTranslate text to a target language
agent(model, instructions, query) -> StringRun an agent with instructions
agentStream(model, instructions, query) -> StringRun an agent with streaming output
chain(steps, input) -> StringExecute a chain of steps
extract(text, schema) -> StringExtract structured data from text
