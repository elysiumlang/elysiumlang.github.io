@elysium/auth 0.1.0

Authentication module — JWT, Password Hashing, Sessions, RBAC, OAuth2, Passkey, API Keys, Multi-tenant.

epm install @elysium/auth

Import

import "#/@elysium/auth" as auth

API

All functions return String results (JSON-encoded where applicable).

JWT

jwtSign(secret, payload) -> StringSign a JWT token with HMAC-SHA256
jwtVerify(secret, token) -> StringVerify a JWT token, returns payload or error
jwtDecode(token) -> StringDecode a JWT token without verification

Session Management

createSession(userId, data) -> StringCreate a new session
getSession(sessionId) -> StringRetrieve session data
destroySession(sessionId) -> StringDestroy a session

Password Hashing

hashPassword(password) -> StringHash a password
verifyPassword(password, hash) -> StringVerify a password against a hash

RBAC (Role-Based Access Control)

checkPermission(user, permission) -> StringCheck user permission
hasRole(user, role) -> StringCheck if user has a role
hasScope(token, scope) -> StringCheck scope in a token
grantRole(user, role) -> StringGrant a role to a user
grantPermission(user, perm) -> StringGrant a permission to a user
revokeRole(user, role) -> StringRevoke a role from a user
revokePermission(user, perm) -> StringRevoke a permission from a user
setRoles(userId, rolesJson) -> StringSet roles from JSON
setPermissions(userId, permsJson) -> StringSet permissions from JSON
checkAccess(userId, resource, action) -> StringCheck access to a resource

OAuth2

oauth2Authorize(clientId, redirectUri, scope) -> StringStart OAuth2 authorization
oauth2Token(code, clientId, clientSecret) -> StringExchange code for tokens
oauth2Refresh(refreshToken, clientId) -> StringRefresh an access token

Passkey / WebAuthn

passkeyRegister(userId, userName) -> StringRegister a passkey
passkeyAuthenticate(userId) -> StringAuthenticate with passkey

Multi-Tenant

tenantContext(tenantId) -> StringSet tenant context
getTenant() -> StringGet current tenant ID
listTenants() -> StringList all tenants
createTenant(tenantId, config) -> StringCreate a new tenant

API Keys

generateApiKey(userId) -> StringGenerate an API key
validateApiKey(apiKey) -> StringValidate an API key
