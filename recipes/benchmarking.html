Benchmarking

Measure wall-clock time with bench / bm blocks. Intermediate

Code

// Fibonacci — a compute-heavy function
func fibonacci(n: Int) -> Int {
    if n < 2 {
        n
    } else {
        fibonacci(n-1) + fibonacci(n-2)
    }
}

func main() {
    let x = 0

    // Use bm (shorthand) or bench (full keyword)
    bm {
        x = fibonacci(30)
    }

    // Use the result to prevent optimization
    let _ = x
}

Expected Output

bench: 0.012345 s

How It Works

  1. bench and bm are synonyms — use whichever you prefer.
  2. The compiler emits LLVM IR with clock_gettime(CLOCK_MONOTONIC, ...) calls around the body block.
  3. A timespec struct is allocated as an [2 x i64] on the stack.
  4. Results are printed to stdout: bench: %.6f s\n via printf.
  5. The body's statements are compiled inline between start/end timing calls, ensuring accurate measurement.
