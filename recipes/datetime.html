Date & Time

Work with unix timestamps across backend and frontend. Beginner

Getting the Current Time

func main() {
    // Current unix timestamp (seconds since epoch)
    let now = datetime.now()

    // Format it
    let formatted = datetime.format(now, "%Y-%m-%d %H:%M:%S")
    print("Current time: ", formatted)

    // Or use fromTimestamp for locale-default formatting
    let str = datetime.fromTimestamp(now)
    print(str)
}

Component Extraction

let now = datetime.now()
let y = datetime.year(now)
let m = datetime.month(now)
let d = datetime.day(now)
let h = datetime.hour(now)
let min = datetime.minute(now)
let s = datetime.second(now)
let wd = datetime.weekday(now)

print(y, "-", m, "-", d)   // e.g. 2026-5-29
print("Day of week: ", wd)      // 0=Sunday, 6=Saturday

Date Arithmetic

let now = datetime.now()
let tomorrow = datetime.addDays(now, 1)
let nextHour = datetime.addHours(now, 1)
let diff = datetime.diffSeconds(tomorrow, now)

print("Seconds until tomorrow: ", diff)   // 86400

Unified Bridge

All datetime APIs use unix timestamps (seconds since epoch, Int) as the universal data bridge. This ensures data flows seamlessly between the C backend and JS runtime:

  • C backend: Uses time(), localtime(), and struct tm operations.
  • JS runtime: Wraps native Date — all methods produce/consume the same unix timestamps.
