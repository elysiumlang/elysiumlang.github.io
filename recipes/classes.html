Classes & Private Modifiers

Object-oriented constructs — classes, constructors, access control, and lazy evaluation. Intermediate

Class with Private Fields

/// Summary: User class with private email field.
class User {
    let name: String
    private let email: String

    init(name: String, email: String) {
        this.name = name
        this.email = email
    }

    private func validate() -> Bool {
        return true
    }

    func greet() -> String {
        return "Hello, " + name
    }
}

Private & Lazy Functions

/// Private — only accessible within the same module
private func helper() -> Int {
    return 42
}

/// Lazy — evaluated on first call, cached thereafter
lazy func expensiveComputation() -> Int {
    return helper() * 2
}

/// Both modifiers together
private lazy func cachedResult() -> String {
    return "computed-lazy"
}

Lazy Variables

/// Lazy let — evaluated on first access
func demoLazy() -> Int {
    lazy let x = expensiveComputation()
    return x   // expensiveComputation() called here
}

Modifier Rules

ModifierApplies ToBehavior
privateFunctions, class fields, class methodsAccess restricted to the defining module/class
lazyFunctions, let variablesEvaluated on first access, cached thereafter
private lazyFunctionsBoth restrictions apply
