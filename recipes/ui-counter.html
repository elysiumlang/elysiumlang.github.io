UI Counter Component

Build a reactive counter with Elysium's declarative UI layer. Intermediate

Counter Component (`.elyx` syntax)

This example uses the .elyx file format — note the render { … } block with XML-style elements. See the UI Guide for a full explanation of the syntax.

counter.elyx
/// Summary: Counter component with increment button.
component Counter {
    state count = 0

    render {
        <Column>
            <Text>Count: {count}</Text>
            <Button label="Increment" onClick={count = count + 1} />
        </Column>
    }
}

Greeting with Conditional Rendering

component Greeting {
    state isLoggedIn = true

    Column {
        if isLoggedIn then
            Text("Welcome back!")
        else
            Button(label: "Login") {
                isLoggedIn = true
            }
    }
}

UI Features Used

How It Works

  • Views are immutable descriptions — no direct DOM manipulation.
  • The runtime diffs and patches the actual UI automatically.
  • Any view that reads a state variable re-renders when that variable changes.
  • The UI layer compiles to platform-native widgets (or a virtual DOM) — it's part of the language, not an external framework.
