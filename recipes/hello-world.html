Hello, World!

Your first Elysium program. Beginner

Code

hello.ely
/// Summary: Greet the user by name.
func greet(name: String) -> String {
    "Hello, " + name + "!"
}

let message = greet("Alice")
print(message)

Run It

elysium run hello.ely
# Output: Hello, Alice!

Explanation

  1. Doc comment — /// Summary: ... is required for production code and provides AI-friendly documentation.
  2. Function definition — func greet(name: String) -> String declares a function with typed parameters and return type.
  3. Implicit return — The last expression in the function body ("Hello, " + name + "!") is automatically returned.
  4. String interpolation — + concatenates strings directly.
  5. print() — Built-in function for bare output (no prefix).
