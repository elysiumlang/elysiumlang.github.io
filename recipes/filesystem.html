Filesystem Operations

Read, write, and manipulate files and directories. Beginner

Code

fs.ely
func main() {
    // Write a file
    fs.writeFile("test_output.txt", "Hello from Elysium fs!")

    // Check if it exists
    let exists = fs.exists("test_output.txt")
    print("File exists: ", exists)

    // Read it back
    let content = fs.readFile("test_output.txt")
    print("Content: ", content)

    // Append more content
    fs.appendFile("test_output.txt", "\nSecond line")

    // Create a directory
    fs.createDir("test_dir")

    // Copy the file
    fs.copyFile("test_output.txt", "test_output_copy.txt")

    // Rename
    fs.rename("test_output_copy.txt", "test_output_renamed.txt")

    // Cleanup
    fs.removeFile("test_output_renamed.txt")
    fs.removeFile("test_output.txt")
    fs.removeDir("test_dir")

    print("fs operations completed\n")
}

Available Functions

FunctionDescription
fs.readFile(path)Read entire file contents
fs.writeFile(path, content)Write content (overwrites)
fs.appendFile(path, content)Append to end of file
fs.exists(path)Check if path exists
fs.copyFile(src, dst)Copy file
fs.rename(old, new)Rename or move
fs.removeFile(path)Delete file
fs.createDir(path)Create directory
fs.removeDir(path)Remove empty directory

Backend Details

  • C backend: Uses stdlib directly — fopen, fgets/fputs, fclose, remove, access, mkdir, rmdir, rename.
  • JS runtime: Maps to Node's native fs module (sync variants for read/write, etc.).
