Functions & Imports

Define reusable functions and import them across files. Beginner

1. Define a Library Module

math.ely
/// Summary: Utility math functions.
func square(x: Int) -> Int { x * x }
func double(x: Int) -> Int { x + x }

2. Import Without Alias

Functions are inlined into the importing module's namespace.

main.ely
/// Summary: Main program that imports math utilities.
import "./math.ely"

let a = 5
let sq = square(a)
let db = double(a)
print(sq)
print(db)

3. Import With Alias

Use import ... as alias to namespace functions. Calls like alias.fn() desugar to alias_fn().

main_alias.ely
/// Summary: Using import-as alias.
import "./math.ely" as math

let a = 5
let sq = math.square(a)
let db = math.double(a)
print(sq)
print(db)

Key Points

  • Imports resolve relative paths from the importing file's directory.
  • Aliased imports prevent name collisions — use them for third-party libraries.
  • Import without alias is convenient for internal modules.
  • Spec blocks can import modules too (see Spec-Driven Testing recipe).
