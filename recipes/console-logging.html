Console Logging

Unified logging with level-specific prefixes. Beginner

Code

console.ely
func main() {
    console.debug("debug message")
    console.info("info message")
    console.warn("warn message")
    console.error("error message")
    console.log("log message")

    // Bare print — no prefix
    print("bare print")

    // Multiple arguments
    let name = "Elysium"
    console.info("Hello, ", name)
    print("Goodbye, ", name)
}

Expected Output

[DEBUG] debug message
[INFO] info message
[WARN] warn message
[ERROR] error message
[LOG] log message
bare print
[INFO] Hello, Elysium
Goodbye, Elysium

How It Works

  • In the C backend: console.* desugars to __console_* builtins emitting LLVM printf calls with the appropriate prefix.
  • In the JS runtime: Maps to native console.* API with environment detection (browser vs Node).
  • print() is always available and outputs with no prefix — useful for program output vs diagnostic messages.
