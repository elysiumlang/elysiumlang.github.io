HTTP Requests

Fetch data from REST APIs using GET, POST, PUT, and DELETE. Intermediate

HTTP Methods

transport_http.ely
func main() {
    // GET — fetch data
    let data = transport.get("https://api.example.com/data")
    print(data)

    // POST — create a resource
    transport.post("https://api.example.com/data",
        "{\"key\":\"value\"}")

    // PUT — update a resource
    transport.put("https://api.example.com/data/1",
        "{\"key\":\"updated\"}")

    // DELETE — remove a resource
    transport.delete("https://api.example.com/data/1")
}

WebSocket

transport_ws.ely
let ws = transport.wsConnect("wss://echo.example.com")
transport.wsSend(ws, "hello from Elysium")
transport.wsClose(ws)

MQTT

transport_mqtt.ely
let client = transport.mqttConnect(
    "mqtt://broker.example.com",
    "elysium-client")
transport.mqttPublish(client, "topic/hello", "world")
transport.mqttSubscribe(client, "topic/#")
transport.mqttDisconnect(client)

Backend Details

  • C backend: HTTP uses curl via snprintf("curl -s -m 10 '%s'", url) + popen. WebSocket & MQTT print stub messages — they require the JS runtime for full support.
  • JS runtime: HTTP via native fetch (Node 18+ / browser), WebSocket via WebSocket API, MQTT via mqtt package.
