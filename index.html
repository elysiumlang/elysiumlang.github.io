Why Elysium?

Designed to make programming feel like writing clear instructions.

🗣️

Natural Syntax

if … then, bc (because), switch … case … then, inline JSON { "key": value }, backtick strings `multi-line` — reads like English.

 🛡️

Type Safety

Full type inference, algebraic data types, generics, Result/Option, union types, and the is operator for runtime type checks.

 ⚡

Concurrency

async/await, parallel blocks with pthreads, channels, select, schedule for timed tasks, and wait for pauses — built right in.

 🧪

Spec-Driven Development

Write spec blocks with feat/expect, todo, question, and bench right alongside your code. Run with ely test.

 🎨

Declarative UI

Components with state, reactive rendering, if … then conditionals, and two-way binding — no framework needed. Use .elyx for XML-syntax render {} blocks.

 📦

Rich Standard Library

Filesystem, networking (HTTP/WebSocket/MQTT), string crypto, regex, datetime, dict, json, math, env, http, auth, and LangChain/LangGraph AI packages.

 🧠

ARC Memory Model

Automatic Reference Counting with weak/unowned, optional only ownership, and unsafe blocks when needed.

 🔧

Full Toolchain

Rust compiler (LLVM backend), npm package with JS runtime, EPM package manager, ely test, ely update, ely migrate, linter, highlighter, and doc generator.

 🤖

AI-Friendly

Enforced /// Summary: doc comments, bc annotations for intent, and well-structured code that AI tools love.

 📋

Inline JSON & Backtick Strings

Create objects with { "key": value } syntax. Multi-line strings with ` — no escaping needed for " characters.

 📡

BLE & Zigbee Packages

Bluetooth Low Energy and Zigbee home automation as pure Elysium packages. Import, scan, control devices — all in .ely.

 ⏰

Schedule & Wait

Cron expressions and friendly intervals ("every 5 minutes", "daily at 08:00"). Pause with wait 1000.

Philosophy

Simplicity

Syntax designed to be easily readable and writable. No cryptic incantations.

Expressiveness

High-level abstractions for concise representation of complex tasks.

Type Safety

Strong typing with clear error messages. Catch bugs at compile time, not runtime.

Minimal Micromanagement

Focus on what you want to do, not on ceremony. The language handles the details.

AI-Friendliness

Well-defined structure, enforced documentation, and bc annotations make it ideal for AI-assisted development.

Todo App in Elysium

A complete, production-style todo application using the declarative UI layer — in just 30 lines.

TodoApp.elyx
/// Summary: A simple todo list application.
component TodoApp {
    state tasks = [String]()
    state newTask = ""

    render {
        <Column padding="20">
            <Text style={{ fontSize: 24, bold: true }}>My Todos</Text>

            <Row>
                <TextField value={newTask} placeholder="Add a task" />
                <Button label="Add" onClick={ only newTask.trim() != ""; tasks = tasks + [newTask]; newTask = "" } />
            </Row>

            if tasks.isEmpty then
                <Text style={{ color: "gray" }}>No tasks yet!</Text>
            else
                <Column>
                    …{tasks.map((task, index) =>
                        <Row>
                            <Text>{task}</Text>
                            <Button label="Delete" onClick={ tasks = tasks.filter((_, i) => i != index) } />
                        </Row>
                    )}
                </Column>
        </Column>
    }
}
